To see the list of users on the track list:
<highlight><tab><symbol>track<end>

To see the track history for a specific player:
<highlight><tab><symbol>track show player<end>

To add a player to the track history:
<highlight><tab><symbol>track add player<end>

To remove a player from the track history:
<highlight><tab><symbol>track rem player<end>

Tracked characters are announced via the source system(tracker)
Make sure you have routes in place to display these messages
where you want to see them. See <a href='chatcmd:///tell <myname> help route'><symbol>help route</a> for more information.
