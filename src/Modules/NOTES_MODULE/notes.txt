<header2>Overview<end>

To show all yours notes:
<highlight><tab><symbol>notes<end>

To show all your notes with reminders:
<highlight><tab><symbol>reminders<end>

<header2>Add or remove notes/reminders<end>

To add a note to your list:
<highlight><tab><symbol>notes add 'message'<end>

To add a note and be reminded about it on logon:
<highlight><tab><symbol>reminder add 'message'<end>

To add a note and be reminded about it on logon only on that char:
<highlight><tab><symbol>reminder addself 'message'<end>

To remove a note from your list:
<highlight><tab><symbol>notes rem 'note id'<end>

<header2>Change reminders<end>

To remove the reminder for a note:
<highlight><tab><symbol>reminders set off 'note id'<end>

To change the reminder for a note to inform only the target character:
<highlight><tab><symbol>reminders set self 'note id'<end>

To change the reminder for a note to inform all alts of the target character:
<highlight><tab><symbol>reminders set all 'note id'<end>
