To roll a number between 1 and a number:
<highlight><tab><symbol>roll 'number'<end>

To roll a number between a start_value and an end_value:
<highlight><tab><symbol>roll 'start_value' 'end_value'<end>

To roll a random value:
<highlight><tab><symbol>roll 'value1' 'value2' 'value3' ...<end>
Note that values must not contain spaces.

To roll multiple random values from a list:
<highlight><tab><symbol>roll 'amount'x 'value1' 'value2' 'value3' ...<end>
Note that values must not contain spaces.

For example, you wanna roll the 2 DB2 infusers for your party:
<highlight><tab><symbol>roll 2x Player1 Player2 Player3 Player4 Player5 Player6<end>

To verify a roll:
<highlight><tab><symbol>verify 'verify number'<end>
